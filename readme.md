# better-touchbaar

my macbook runs nixos, btw.

the touchbar on macbooks is pretty nice, and [tiny-dfr](https://github.com/AsahiLinux/tiny-dfr) on linux isn't too bad, but i wanted something prettier. so i made this!
it probably won't fit your needs (at least not right now) and it is not finished, but i will add real configuration and such soon.
pictures and an actual usage guide will be here soon!

i have "borrowed" a lot of code from tiny-dfr (in fact, display.rs is completely verbatim), hence the MIT license rather than the AGPL i normally go with. i think i've done the license correctly, please let me know if it is wrong!
